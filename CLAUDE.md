## Project Overview

DirectiveFilms is a Vite + React Router v7 (framework mode) web application. All 6 routes are static client-rendered pages pre-rendered to HTML at build time for SEO. Built with React 19, TypeScript, and the React Compiler (via Babel plugin).

## Development Commands

```bash
# Start development server
pnpm dev
# Runs on http://localhost:3001

# Type checking (runs typegen first)
pnpm type-check

# Linting
pnpm lint          # Check for issues
pnpm lint:fix      # Auto-fix issues

# Build (pre-renders all routes to build/client/)
pnpm build

# Full validation
pnpm check         # type-check + build

# Serve production build locally
pnpm start
```

## Architecture

### Vite + React Router v7 Framework Mode

- **Config**: `react-router.config.ts` ‚Äî `ssr: false`, `prerender: true`, `appDirectory: "src"`
- **Vite config**: `vite.config.ts` ‚Äî React Router plugin + `@vitejs/plugin-react` with React Compiler babel plugin
- **Root**: `src/root.tsx` ‚Äî App shell with `<Layout>` (html/head/body), fonts, globals.css, cursor wrapper
- **Routes**: `src/routes.ts` ‚Äî Defines 6 routes pointing to route modules in `src/routes/`
- **Route modules**: `src/routes/*.tsx` ‚Äî Thin wrappers that export `meta()` + default component
- **Generated types**: `.react-router/types/` ‚Äî Auto-generated by `react-router typegen` (run via `pnpm type-check`)

### Fonts

Self-hosted via `@fontsource` (imported in `src/root.tsx`):
- **Playfair Display** (700, 800, 900) ‚Äî Display/heading font
- **Inter** (400, 500, 600) ‚Äî Body font
- **Space Mono** (400) ‚Äî Monospace font

CSS variables: `--font-display`, `--font-body`, `--font-mono`

### Design System

- **Design tokens**: `src/tokens.js` ‚Äî colors, typography, spacing, layout, etc.
- **CSS Custom Properties**: Layout tokens in `src/globals.css`
- **Design system page**: `/design-system` ‚Äî Visual showcase of all tokens

### Animation System

- **Framer Motion** ‚Äî Scroll tracking, viewport detection, springs
- **Lenis** ‚Äî Smooth scroll
- **split-type** ‚Äî Text splitting for typography animations
- See `AWARD-WINNING-ANIMATIONS.md` for full documentation

### Deployment

- **Vercel** ‚Äî Static deployment via `vercel.json`
- Build output: `build/client/` with pre-rendered HTML for all routes
- SPA fallback via `__spa-fallback.html` rewrite

### TypeScript

- Strict mode enabled
- Path alias: `@/` ‚Üí `./src/`
- CSS modules typed via `vite/client` reference
- React Router types generated in `.react-router/types/`
